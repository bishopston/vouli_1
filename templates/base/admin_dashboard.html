Αναλυτικά στοιχεία για τα σχολεία που έχουν πραγματοποιήσει εγγραφή στην εφαρμογή επισκέψεων και τις κρατήσεις τους
Σχολικές Μονάδες
Διαχείριση του μητρώου σχολικών μονάδων πρωτοβάθμιας / δευτεροβάθμιας εκπαίδευσης όλης της Ελλάδας
Σχολεία Διαχειριστή

Περίοδοι Κρατήσεων
Καθορισμός και διαχείριση περιόδων κατά τις οποίες μπορεί να πραγματοποιηθούν κρατήσεις για κάθε μία από τις περιόδους επισκέψεων
Περίοδοι Επισκέψεων

Καθορισμός και διαχείριση χρονοθυρίδων κατά τις οποίες επιτρέπονται επισκέψεις στη διάρκεια μιας περιόδου επισκέψεων
Κατά Εξαίρεση Χρονοθυρίδες
Καθορισμός και διαχείριση επιτρεπτών χρονοθυρίδων για επισκέψεις σε συγκεκριμένες ημέρες μιας περιόδου επισκέψεων
Διαχείριση Κρατήσεων
Επισκόπηση και διαχείριση κρατήσεων για την εκάστοτε περίοδο που είναι διαθέσιμη για δημιουργία κρατήσεων επίσκεψης
Ημερολόγιο Κρατήσεων
Επισκόπηση ημερολογίου κρατήσεων και διαθεσιμότητας ημερών για συγκεκριμένη περίοδο ενδιαφέροντος
Κρατήσεις Διαχειριστή
Επισκόπηση και διαχείριση κρατήσεων που έχουν δημιουργηθεί από τον διαχειριστή για λογαριασμό σχολείων
Ιστορικό Κρατήσεων
Επισκόπηση συνολικού ιστορικού κρατήσεων και δυνατότητα φιλτραρίσματος ανά έτος, περίοδο, περιφερειακή διεύθυνση, σχολείο
Στατιστικά Στοιχεία
